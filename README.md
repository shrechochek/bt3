# Сайт с олимпиадными заданиями по биологии
> На сайте можно создавать олимпиадные задания по биологии, тесты из заданий, а также узнавать результаты вашиз учеников

---

<h2 id="download"> Установка </h2>

1. Клонируйте репозиторий:

```bash
git clone https://github.com/shrechochek/bt3
```

2. установите библиотеки

```bash
pip install requirements.txt
```

3. перейдите в папку с файлом `app.py`

```bash
cd bt3
```

4. запустите программу:

```bash
pyhton app.py
```

после этого сайт будет доступен по адресу:

`http://127.0.0.1:9292/`

---

для того, чтобы создать админа, вам нужно создать пользователя с `username:` **admin** и у него будет доступ к админ панеле
